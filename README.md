# Engine

Engine - 3D graphics and physics engine 

***

## The Plan
The plan for this engine is to create a multi-platform fast 3D graphics and physics engine.

### Supported graphics APIS:
- OpenGL
- Direct3D

### Supported platforms:
- Windows

### Features:
- Text Rendering
- Instancing
- Batch Rendering
- Models Rendering

***
### Main features to come:
- Vulkan support
- Metal support
- OpenGL ES support
- Linux support
- Audio system
- Terrain rendering
